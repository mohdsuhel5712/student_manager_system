## 📌 Description
This is a command-line based Student Manager application built using Python.  
It allows users to manage student records using a simple menu-driven interface.  
Users can add, view, and delete student details easily through the terminal.

---

## 🚀 Features
- ➕ Add new student  
- 📋 View all students  
- ❌ Delete student  
- 🔁 Menu-driven program  
- 🧼 Clean and simple output  

---

## 🛠️ Technologies Used
- Python (Core Concepts: List, Dictionary, Functions, Loops)

---

## ▶️ How to Run
1. Install Python on your system  
2. Download or clone this project  
3. Open terminal in project folder  
4. Run the program using:

   python main.py
